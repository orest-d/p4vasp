decode(text, struct=None)

source code 

Decode selection from text. If text contains a *selection* in the selection language, and (optionally) struct contains the structure, this function returns a list of indexes of the selected atoms. A valid Structure with AtomInfo is needed if the species indexes and/or elements are used in the selection string.

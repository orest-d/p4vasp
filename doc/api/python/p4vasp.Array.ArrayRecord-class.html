Class ArrayRecord

source code

A wrapper class to the Array record, that allows a more convenient handling of records. It is returned by *Array.getRecord()* .

Array is stored in *self.array_* and record in *self.record* , however, the use *getArray()* and *getRecord()* is recomended.

Array fields can be acessed using:

a. dictionary convention (e.g. rec["field"]),

b. list convention (by field numbers) (e.g. rec[1])

c. as attributes (e.g. rec.field) .

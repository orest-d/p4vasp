__init__(self, name, attribute=None, encode=1, tag=1, tagattr=0)
(Constructor)

source code 

Attribute profile constructor

*name* - name, by which is the attribute identified in the xml file

*attribute* - name, by which is the attribute identified in object (class attribute) If not specified, it is the same as name.

*encode* - if true, the attribute will be string-encoded

*tag* - if true, *name* will be used as a tag name

*tagattr* - if true, information will be stored in form of a xml attribute

Set attribute value from object encoded as a string. By default no decoding is defined, the string-encoded value is just passed. This is suitable for simple strings, but it needs to be overriden for other objects.

indent(s, istring)

source code 

Intend *s* by *istring*. If istring is an iteger, istring is changed to istring*INDENT (INDENT contains two whitespaces by default). Then *istring* is added to the beginning of every line of *s*.

guessType(x)

source code 

Try to guess the type from the python type. This maps *float* to *FLOAT_TYPE*, *int* to *INT_TYPE* and everything else to *STRING_TYPE* .

retype(x, t)

source code 

Return x converted to a scalar of type t.

For *LOGICAL_TYPE* this converts

* integers to (*x* != 0),

* ".TRUE." and strings starting with "T, "1", "Y" to 1 (case insensitive),

* other values to 0.

retypeVec(x, t)

source code 

Return x converted to a vector of type t. This is the same as *retype()* function applied to all elements of *x*.

xmlrepr(x, t='float', format=None, float_format=None, string_format=None, logical_format=None, int_format=None)

source code 

Represent *x* of type *t* as a string using *format*, or if *format* is None, using default formats ( *xxx_format* ) for particular type. If the *format* (or *xxx_format*) is *None*, *str()* function is used for converting.

LOGICAL_TYPE is first converted to "T" or "F".

If *x* is an array, the format is applied ao all elements and the result is joined using the whitespace as a separator.

getTextFromElement(elem)

source code 

Extracts all texts and CDATA sections from the element *elem* and returns them joined as one string. Note: The returntype is the ordinary string, not unicode.

source code

isArray(x)

source code 

Returns true for

* python *lists*,

* *tuples*,

* *UserList* instances,

* *Vector* instances (from the matrix library),

* *array* objects from the *numpy* library.

arrayShape(l)

source code 

Returns the *shape* (i.e. list of dimensions) of the array l. For *Numeric* array object this function simply returns array.shape. For other objects a tuple containing (len(l), len(l[0]), len(l[0][0]) ...) is returned.

Class Dictionary

source code

The basic dictionary object, it can be created from arbitrary xml element node containing <i> (item) and <v> (vector) nodes with specified "name" attribute. The type of the dictionary (usually a tag-name, e.g. "incar") is stored in the variable "name".

XML element node from which the *Dictionary* was constructed is stored in the *element* attribute.

Dictionary behaves like ordinary python dictionary (inherits from UserDict).

Types of values are kept in the *self.type* dictionary, but *getFieldType()* and *setFieldType()* should be used instead of direct access.
